Monitoramento Inteligente de Rotas de Ônibus

Este projeto consiste em uma aplicação web desenvolvida com **HTML**, **CSS** e **JavaScript** para simular o monitoramento de rotas de ônibus. O objetivo é fornecer uma interface interativa onde usuários podem visualizar o movimento de ônibus em um mapa e obter previsões de chegada.

A aplicação utiliza uma biblioteca de mapas (você pode integrar **Leaflet.js** ou **Google Maps API** aqui) para exibir as rotas e os pontos de parada. O movimento dos ônibus é simulado utilizando lógica **JavaScript**, que atualiza a posição dos veículos em intervalos regulares, dando a impressão de um acompanhamento em tempo real.

Funcionalidades incluem:

  • Visualização de rotas e paradas em um mapa interativo.
  • Simulação de localização de ônibus se movendo ao longo das rotas.
  • Previsão de tempo de chegada dos ônibus nas paradas (simulada).
  • Possibilidade de buscar rotas específicas.

Este projeto demonstra a capacidade de integrar APIs externas (se você usar uma API de mapas), manipulação de dados em tempo real (simulada), e criação de interfaces de usuário dinâmicas, elementos essenciais para aplicações web modernas.

Sistema de Gerenciamento para Abrigos de Animais

Este projeto é um sistema de gerenciamento simples, desenvolvido para auxiliar abrigos de animais na organização e acompanhamento de seus residentes. A interface é construída com **HTML** e **CSS**, e a lógica de gerenciamento (cadastro, exibição, filtragem) é implementada com **JavaScript**.

O sistema permite registrar informações detalhadas sobre cada animal, como nome, espécie, raça, idade, histórico de saúde e status de adoção. Ele oferece uma visão clara dos animais disponíveis, facilitando o processo de adoção e a gestão interna do abrigo.

Funcionalidades incluídas:

  • Cadastro de novos animais com formulários interativos.
  • Visualização de uma lista ou galeria de animais com seus respectivos detalhes.
  • Atualização do status de adoção dos animais.
  • Possibilidade de filtrar animais por espécie ou status.
  • Gerenciamento de informações de contato e doações para o abrigo (simulado).

Este projeto demonstra habilidades em manipulação do DOM, gerenciamento de estados (armazenamento local ou em arrays JavaScript), e criação de interfaces amigáveis para aplicações de dados. É um excelente exemplo de como a programação pode ser utilizada para causas sociais.

Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma coleção pessoal de livros favoritos. O objetivo é criar um ambiente virtual onde seja possível compartilhar seus livros preferidos, fornecendo uma descrição breve de cada um, incluindo o autor, o ano de publicação e uma opção de compra.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de livros em uma seção flexível (.livros). Cada livro é destacado em seu próprio contêiner (.livro), mostrando uma imagem da capa, detalhes do livro e um link de compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração da coleção de livros.
